RESUMEN PARA NUEVO AGENTE - Sistema Inbox Social Media
Proyecto
Sistema de gestión de inbox para redes sociales que se integra con Metricool API para centralizar DMs y comentarios de múltiples plataformas (Instagram, Facebook, TikTok, LinkedIn, YouTube, Google Business).

Estado Actual
✅ Completado: Instagram (141 msgs), LinkedIn (2 msgs), TikTok (2 msgs) = 145 mensajes totales
🎯 Pendiente: Mapear datos de Facebook
Arquitectura de Sincronización
1. Backend - Conexión con Metricool:

Archivo: server/services/metricool.ts

Autenticación: Header X-Mc-Auth: {METRICOOL_USER_TOKEN}
Parámetros: userId (env var), blogId (de la marca)
Endpoints:
/api/v2/inbox/conversations → DMs (Facebook, Instagram)
/api/v2/inbox/post-comments → Comentarios públicos
2. Backend - Mapeo de datos:

Archivo: server/routes.ts (líneas ~420-520)

Función normalizePlatform(): Convierte nombres de providers (ej: TIKTOKBUSINESS→tiktok)
Para cada plataforma, extrae datos del JSON de Metricool y los mapea a nuestra estructura:
author → Nombre del usuario
authorAvatar → URL de foto de perfil
content → Texto del mensaje/comentario
timestamp → Fecha/hora
sourceUrl → Link al post original
rawData → JSON completo guardado para debugging
3. Frontend - Adaptador:

Archivo: client/src/lib/api.ts

Función adaptMessage(): Convierte type: 'conversation' → 'dm' para el UI
Mantiene compatibilidad entre schema de DB y tipos del frontend
Estructura de Datos en DB (shared/schema.ts)
messages: {
  id, brandId, platform, type ('conversation'|'comment'),
  author, authorAvatar, content, timestamp, status,
  metricoolId (único), sourceUrl, rawData (jsonb)
}

Patrón de Mapeo (ya implementado para Instagram/LinkedIn/TikTok)
El JSON de Metricool viene con estructuras diferentes por plataforma. Ejemplo de cómo se mapea:

// Para comentarios con participants[]
if (['linkedin', 'tiktok', 'instagram'].includes(provider)) {
  const participant = item.participants?.[0];
  author = participant?.name || 'Unknown';
  authorAvatar = participant?.imageProfileUrl || '';
}
// Para DMs de Instagram (participants[] + from field)
const fromId = conv.from;
const sender = conv.participants?.find(p => p.id === fromId);
author = sender?.name || 'Unknown';

# KICKOFF: Implementación de Autenticación Social con Replit Auth

## Objetivo
Implementar login social con Google, GitHub, Apple y Twitter (X) usando Replit Auth nativo, manteniendo compatibilidad con el sistema de autenticación email/password existente.

## Contexto del Proyecto
Este es un Social Media Inbox Management System (Repliyo) integrado con Metricool. Ya tiene autenticación funcional con email/password, y ahora necesitamos agregar OAuth social.
